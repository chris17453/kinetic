Required Dependencies

This design system is built on top of Bootstrap 5 and Font Awesome 6. Both must be included in every page.

CDN Links (Required)

<!-- Bootstrap 5 CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Font Awesome 6 -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

<!-- Bootstrap 5 JS (at end of body) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

Component Rules

Buttons

  • Always use .btn base class
  • Add variant: .btn-primary, .btn-secondary, .btn-outline, .btn-ghost, .btn-success, .btn-danger
  • Add size modifier if needed: .btn-sm, .btn-lg
  • For icon-only buttons: .btn-icon
<button class="btn btn-primary">Primary</button>
<button class="btn btn-outline btn-sm">Small Outline</button>
<button class="btn btn-icon btn-ghost"><i class="fas fa-cog"></i></button>

Status Badges

  • Use .badge base class
  • Add status: .badge-queued, .badge-processing, .badge-completed, .badge-failed, .badge-default
  • Small variant: .badge-sm
<span class="badge badge-completed">Completed</span>
<span class="badge badge-sm badge-processing">Processing</span>

Forms

  • Wrap each field in .form-group
  • Use .form-label for labels
  • Use .form-input, .form-select, .form-textarea for inputs
  • Use .form-row for two-column layouts
  • Use .form-checkbox for checkbox labels

Cards

  • Use .card container
  • Optional: .card-header, .card-body, .card-footer

Alerts

  • Use .alert base class
  • Add type: .alert-success, .alert-warning, .alert-error, .alert-info
  • Include .alert-icon and .alert-content children

Dropdown Menus

  • Wrap in .dropdown container
  • Use .dropdown-menu for the popup (add .show class to display)
  • Use .dropdown-item for menu items
  • Use .dropdown-divider for separators
  • Use .dropdown-header for section headers
  • Add .danger to .dropdown-item for destructive actions
<div class="dropdown">
  <button class="btn btn-secondary" onclick="toggleDropdown('myMenu')">
    Actions <i class="fas fa-chevron-down"></i>
  </button>
  <div class="dropdown-menu" id="myMenu">
    <div class="dropdown-header">Options</div>
    <a href="#" class="dropdown-item"><i class="fas fa-edit"></i> Edit</a>
    <a href="#" class="dropdown-item"><i class="fas fa-copy"></i> Copy</a>
    <div class="dropdown-divider"></div>
    <a href="#" class="dropdown-item danger"><i class="fas fa-trash"></i> Delete</a>
  </div>
</div>

Do's and Don'ts

  • DO include Bootstrap 5 CSS and JS in every page
  • DO include Font Awesome 6 for icons
  • DO use CSS variables for all colors, spacing, and sizing
  • DO use semantic class names (e.g., .btn-primary not .blue-button)
  • DO use Font Awesome icons with proper prefixes (fas, far, fab)
  • DO test both light and dark modes
  • DON'T use hardcoded hex colors (e.g., #0066cc)
  • DON'T use arbitrary pixel values for spacing
  • DON'T override component styles with inline styles
  • DON'T use !important unless absolutely necessary
  • DON'T mix Bootstrap's default button classes with custom .btn-* classes
